README
--------

This is an Activity module for Moodle 2.x that allows users to create an Ishikawa Diagram (a variation of it, in fact).

Main Features:

* Teacher can define the number of columns and lines for diagram Causes and Effects
* Support grades
* Support groups
* Backup and restore

Dependencies:

* This module depends on Image Magick for generate the diagrams. You MUST install php5-imagick package.
